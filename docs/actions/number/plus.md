# plus

- Alias: `+`
- Syntax: `<number|array> plus <number|array>`

Performs an [addition](https://en.wikipedia.org/wiki/Addition) on numbers and numbers within arrays:

    print_line 1 + 2; # 3
    print_line [1 2] + 3; # 4 5
    print_line 3 + [2 1]; # 5 4
    print_line [1 2] + [3 4]; # 4 6
