# print

Aliases: `output`, `write`, `echo`, `say`, `<`

Prints the given [string](../constructs#string) to the console and returns it.

    > print "Hello, world!";
    < Hello, world!
    = Hello, world!

## See also

- [print_line](print_line): Does the same as print, except that it appends `crlf` to the string.
