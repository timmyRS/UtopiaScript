# array

- Aliases: `arr`, `[...]`
- Syntax: `array [<key> =] <value> ...`

Arrays are collections of values:

```
array 1 2 3
```

and the values may even be given keys to identify their meaning:

```
array name: "John"
      age = 30
;
```

## See also

- [range](range): Creates an array containing a range of numbers or characters.
