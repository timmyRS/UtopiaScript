# Code Golfing

## Noteworthy aliases

- Statements
    - `=` = `return`
    - Conditional
        - `?` = `if`
        - `?!` = `unless`
        - `@` = `while`
        - `@!` = `while_not`
    - Declaration
        - `.` = `local`
        - `,` = `final`
        - `:` = `global`
        - `!` = `const`
    - Standard Input/Output (STDIO)
        - `<` = `print`
        - `<<` = `print_line`
        - `>` = `read`
- Variable actions
    - `=` = `equals`
    - `!=` = `not_equals`
    - Strings
        - `^` = `to_upper_case`
        - `v` = `to_lower_case`
    - Numbers
        - `^` = `pow`
        - `!` = `factorial`
        - `%` = `mod`
        - `c` = `ceil`
        - `r` = `round`
        - `f` = `floor`
    - Arrays
        - `@` = `for_each`
- Constants
    - `_` = `null`
