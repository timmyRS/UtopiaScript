# Code Golfing

## Noteworthy aliases

- Statements
    - `=` = `return`
    - Conditional
        - `?` = `if`
        - `?!` = `unless`
        - `@` = `while`
        - `@!` = `while_not`
        - `|` = `otherwise`
    - Declaration
        - `.` = `local`
        - `,` = `final`
        - `:` = `global`
        - `!` = `const`
    - Standard Input/Output (STDIO)
        - `<` = `print`
        - `<<` = `print_line`
        - `>` = `read`
    - Variables
        - `[...]` = `array ...`
        - `*` = `function`
- Variable actions
    - `=` = `equals`
    - `!=` = `not_equal_to`
    - `s` = `to_string`
    - Strings
        - `^` = `to_upper_case`
        - `v` = `to_lower_case`
    - Numbers
        - `+` = `plus`
        - `-` = `minus`
        - `*` = `times`
        - `/` = `divided_by`
        - `^` = `pow`
        - `%` = `mod`
        - `!` = `factorial`
        - `c` = `ceil`
        - `r` = `round`
        - `f` = `floor`
    - Arrays
        - `@` = `for_each`
